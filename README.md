# MSMP Core

Core plugin of MSMP.

## Installation

Only gradle is supported.

```gradle
repositories {
    maven {
        name = 'jitpack'
        url = 'https://jitpack.io'
    }
}

dependencies {
    implementation 'com.github.msmp:msmp-core:VERSION'
}
```

## Technologies

- Kotlin 1.8.21
- Gradle 7
- Spigot for Minecraft 1.19.4
